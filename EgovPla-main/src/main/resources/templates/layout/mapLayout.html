    도움말
    - 트래픽 / 인구 << 사람들이 많이 다니는 곳(트래픽) 사람들이 많이 사는 곳(인구)중 중요한 곳으로 옮겨주세요.(기본값 0.5 : 0.5)
    - 반복수 << 계산을 몇번 반복할지 정해주세요 높을수록 정확하지만 오래걸리고 낮을수록 빠르지만 정확도가 떨어집니다.(기본값 10000)
    - 새로 만들 와이파이 개수 << 새로 설치할 와이파이 개수를 정합니다. 정해진 개수만큼 화면에 표출합니다.(기본값 5)

    도움말
    - 새로 만들 도서관 개수 << 새로 만들 도서관 개수를 입력합니다.(기본값 1)
    - 도서관 / 버스정류장 << 더 중요시하는 쪽으로 옮겨 주세요 결과에 반영됩니다.(기본값 0.7 : 0.3)
    - 표출할 상위 개수 << 추천 점수가 높은 도서관부터 설정한 숫자의 개수만큼 화면에 보여줍니다. (기본값 5)

    - 새로 만들 그늘막 개수 << 새로 만들 그늘막 개수를 입력합니다.(기본값 1)
    - 커버 가능한 크기 << 그늘막이 시원함을 제공하는 거리를 의미합니다. 값을 키우면 넓은 지역을 커버하지만, 효과가 약해질 수 있습니다. 값을 줄이면 좁은 지역에 집중해서 설치가능합니다. (기본값 1.0km)
    - 온도 / 접근성 << 더 중요시하는 쪽으로 옮겨 주세요 결과에 반영됩니다. 온도가 높을수록 더운곳에 집중하고 접근성이 높을수록 사람들의 쉽게 올 수 있는 곳을 중요시합니다.(기본값 0.5 : 0.5)
